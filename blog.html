Discuss the scope of var, let, and const

var variable is accessible from anywhere within the function in which it is declared, including nested functions.

The let is used to declare a variable that has block scope. let variable is only accessible from within the block in which it is declared.

The const is used to declare a constant variable. A constant variable is a variable whose value cannot be changed after it is initialized.

Tell us the use cases of null and undefined

Null is used to explicitly represent the absence of a value. For example, we can use null to represent a missing value in an object or an array.

Undefined is used to represent the absence of a value that has not been explicitly assigned. For example, if we declare a variable but do not assign it a value, the value of that variable will be undefined

What do you mean by REST API?

REST API stands for Representational State Transfer Application Programming Interface. It is an architectural style for designing and developing web services. REST APIs are based on the HTTP protocol and use a set of standard HTTP methods to perform operations on resources.

The four main HTTP methods used in REST APIs are:

  • GET: Used to retrieve a resource.
  • POST: Used to create a new resource.
  • PUT: Used to update an existing resource.
  • DELETE: Used to delete a resource.

REST APIs are a powerful tool that can be used to build a wide variety of applications. If you are developing a web service, REST API is a good option to consider.
